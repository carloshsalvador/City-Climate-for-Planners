# City Climate for Planners

Prototype Streamlit app for climate-aware urban planning.

## Local run

```bash
streamlit run app/frontend/streamlit_app.py
```

## Deploy on Streamlit Community Cloud

- Connect the GitHub repository.
- Use `app/frontend/streamlit_app.py` as the main file.
- Keep `requirements.txt` at the repository root.
- Add secrets in the Streamlit Cloud dashboard if needed.
